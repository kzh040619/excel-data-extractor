## 📖 产品简介

Excel数据提取工具是一款基于AI的智能Excel查询和导出工具，专为HR部门设计，让员工数据查询变得简单高效。

### 核心优势
- 🤖 **AI智能查询** - 自然语言提问，无需学习复杂语法
- ⚡ **快速导出** - 一键生成Excel文件
- 🔐 **数据安全** - 自动过滤敏感信息
- 📊 **灵活筛选** - 支持多条件组合查询

## 🚀 快速开始

### 系统要求
- Windows 10/11 或 macOS
- Python 3.10+
- Node.js 16+
- Chrome/Edge浏览器

### 安装步骤

**1. 下载项目**
```bash
# 访问GitHub下载
https://github.com/你的用户名/excel-data-extractor

# 或使用git克隆
git clone https://github.com/你的用户名/excel-data-extractor.git
```

**2. 启动后端（终端1）**
```bash
cd excel-data-extractor/backend
pip install -r requirements.txt
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

**3. 启动前端（终端2）**
```bash
cd excel-data-extractor/frontend
npm install
npm run dev
```

**4. 访问系统**
打开浏览器访问：http://localhost:5173

## 🎯 功能说明

### 功能一：字段导出（第一栏）
**适用场景**：已知筛选条件，需要精确导出

**特点**：
- 支持多字段组合筛选
- 支持比较运算符（<, >, =）
- 支持模糊匹配（包含）
- 自动过滤敏感字段

**导出字段**：可选择需要导出的列，默认导出所有列

---

### 功能二：信息查询（第二栏）
**适用场景**：快速查看个人某个字段信息

**格式**：`姓名 字段名`

**示例**：
- `张三 hrbp` → 显示张三的HRBP
- `李四 合同主体` → 显示李四的合同主体

---

### 功能三：AI智能查询（第三栏）⭐
**适用场景**：自然语言提问，智能导出

**特点**：
- 自动理解查询意图
- 支持数量限制
- 支持排序（最近、最早）
- 自动生成Excel文件

**支持的AI服务商**：
- DeepSeek（推荐，有免费额度）
- OpenAI
- 通义千问

## 📚 操作指南

### 一、首次配置

#### 1.1 上传Excel文件
1. 点击顶部 **"上传"** 按钮
2. 选择Excel文件（.xlsx或.xls格式）
3. 系统自动解析工作表和列名

#### 1.2 配置AI（仅首次使用）
1. 点击右上角 **"配置API"** 按钮
2. 选择服务商：**DeepSeek**（推荐）
3. 获取API Key：
   - 访问 https://platform.deepseek.com
   - 注册/登录账号
   - 创建API Key（免费额度够用）
4. 将API Key粘贴到配置框
5. 点击 **"保存配置"**

✅ 配置完成后，AI助手图标显示"✅ AI助手已就绪"

---

### 二、使用字段导出（精确查询）

#### 2.1 基本操作
1. 在对应字段输入框填入筛选条件
2. 点击 **"预览结果"** 查看筛选结果
3. 确认无误后点击 **"导出Excel"**
4. 点击下载链接保存文件

#### 2.2 筛选语法

| 输入方式 | 说明 | 示例 |
|---------|------|------|
| `值` | 精确匹配 | `张三` |
| `<日期` | 小于该日期 | `<2024-12-31` |
| `>数字` | 大于该数值 | `>100` |
| `>=日期` | 大于等于 | `>=2024-01-01` |
| `包含文本` | 模糊匹配 | `包含电商` |

#### 2.3 操作示例

**示例1：查询电商部门员工**
```
所属部门：电商
```

**示例2：合同即将到期**
```
劳动合同/协议结束日期：<2024-12-31
员工状态：在职
```

**示例3：特定HRBP的员工**
```
HRBP姓名：张三
```

#### 2.4 导出字段选择
- 勾选需要导出的列
- 点击 **"全选"** 导出所有可用列
- 点击 **"清空"** 取消所有选择
- 未勾选任何列时，默认导出所有列

### 四、使用AI智能查询（最推荐）⭐

#### 4.1 基本使用
1. 在输入框输入自然语言问题
2. 点击 **"发送"** 或按 `Enter`
3. AI自动分析并生成Excel
4. 点击 **"下载导出文件"** 保存

#### 4.2 查询示例

##### 场景1：离职人员查询
```
最近离职的5个人
2023年离职的员工有哪些
离职人员统计
```

##### 场景2：部门查询
```
找出电商部门的员工
技术部有多少人
查询市场部的所有在职员工
```

##### 场景3：合同到期提醒
```
合同3个月内到期的员工
2024年底合同到期的人员
无固定期限合同的员工
```

##### 场景4：地域查询
```
武汉地区的员工
工作地点在北京的人员
社保在上海缴纳的员工
```

##### 场景5：综合条件
```
找出电商部门最近离职的10个人
武汉地区合同即将到期的员工
技术部2023年入职的员工
```

#### 4.3 高级技巧

**技巧1：数量限制**
```
最近5个、前10条、最新3名
```
AI自动识别数量并限制返回结果

**技巧2：时间排序**
```
最近离职 → 按离职日期降序
最早入职 → 按入职日期升序
```

**技巧3：多轮对话**
系统保留最近6轮对话，可以继续追问：
```
你：找出电商部门的员工
AI：已导出7条记录
你：只要武汉的
AI：已导出3条记录
```

## ❓ 常见问题

### Q1: 提示"请先配置大模型API"？
**A**: 点击右上角"配置API"按钮，填入DeepSeek的API Key后保存。

**获取步骤**：
1. 访问 https://platform.deepseek.com
2. 注册账号并登录
3. 点击"API Keys" → "Create new secret key"
4. 复制Key粘贴到系统中

---

### Q2: AI查询结果不准确？
**A**: 
- 系统会自动智能匹配，如"电商"会匹配"电商部"、"电商事业部"
- 日期筛选自动排除无固定期限（2999年）
- 如果仍不准确，尝试更明确的描述

---

### Q3: 导出的Excel缺少某些列？
**A**: 系统自动过滤以下敏感信息：
- 身份证号、证件号码
- 银行账号、银行卡号
- 电话号码、邮箱
- 家庭住址、户籍地址
- 紧急联系人信息

如需导出，需要修改后端配置文件。

---

### Q4: 日期筛选包含了2999年的记录？
**A**: 
- 检查字段名是否正确
- 使用完整字段名：`劳动合同/协议结束日期`
- 而非简写：`合同结束日期`

---

### Q5: "最近离职的5个人"查询不对？
**A**: 
- 系统自动按离职日期降序排列
- 如果离职日期为空，会排在最后
- 确认Excel中的离职日期字段有数据

---

### Q6: API调用失败或超时？
**A**: 
1. 检查网络连接
2. 确认API Key是否正确
3. DeepSeek服务可能需要稳定网络
4. 检查API额度是否用完

---

### Q7: 如何查看API使用量？
**A**: 
1. 访问 https://platform.deepseek.com
2. 登录账号
3. 查看"Usage"页面

---

### Q8: 能否支持多个Excel文件？
**A**: 
- 当前版本支持单文件查询
- 切换文件：上传新文件或从"选择历史文件"下拉框选择

## ⚠️ 注意事项

### 数据安全
1. ✅ **本地存储** - 所有数据存储在本地，不上传云端
2. ✅ **敏感过滤** - 自动过滤敏感字段
3. ✅ **API安全** - LLM调用仅发送查询条件，不发送原始数据
4. ⚠️ **API Key保护** - 不要分享你的API Key
5. ⚠️ **文件备份** - 定期备份上传的Excel文件

### 使用建议
1. 📌 首次使用建议使用测试数据
2. 📌 确认查询结果后再分享给他人
3. 📌 定期清理导出的临时文件（backend/data/exports/）
4. 📌 AI查询建议先用简单问题测试

### 系统限制
- 单次查询最大返回50条预览记录
- 导出Excel无数量限制
- AI对话保留最近6轮历史
- 上传文件建议小于50MB
