## ⚡ 启动步骤

### 1. 启动后端（终端1）
```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

### 2. 启动前端（终端2）
```bash
cd frontend
npm install
npm run dev
```

### 3. 访问系统
打开浏览器访问：http://localhost:5173
