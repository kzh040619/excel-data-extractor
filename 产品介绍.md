# Excel 数据提取工具 - 产品介绍

## 🎯 这个工具能帮你做什么？

作为 HR 或数据分析师，你是不是经常遇到这样的情况：

● 🔍 同事突然问"帮我查一下张三的 HRBP 和合同到期时间"，你得在 Excel 里翻半天  
● 📊 每月要导出特定部门的人员名单，手动筛选耗时费力  
● ⏰ 合同到期提醒要一个个核对日期，生怕漏掉谁  
● 📈 领导要"最近离职的10个人"，你还得先筛选再排序再导出  
● 🔐 导出数据时担心泄露身份证、银行卡等敏感信息  

**这个工具就是来解决这些烦恼的！**

---

## ✨ 三种查询方式，总有一种适合你

### 方式1️⃣ AI智能查询（最推荐）⭐
**只需要用自然语言提问，系统自动理解并导出Excel！**

你可以这样说：
```
"找出最近离职的5个人"
"查询电商部门的员工"
"合同3个月内到期的员工有哪些"
"武汉地区2023年入职的员工"
```

系统自动帮你：
✅ **智能理解意图** - 自动识别筛选条件、排序规则、数量限制  
✅ **模糊匹配字段** - 说"合同结束日期"自动匹配"劳动合同/协议结束日期"  
✅ **智能排序** - "最近离职"自动按离职日期降序排列  
✅ **一键导出** - 直接生成Excel文件供下载  

---

### 方式2️⃣ 字段筛选（精确查询）
**适合你已经明确知道筛选条件的情况**

可视化界面，直接输入条件：
```
所属部门: 电商
劳动合同/协议结束日期: <2024-12-31
员工状态: 在职
```

支持多种操作符：
- `<日期` - 小于该日期（自动排除2999年无固定期限）
- `>数字` - 大于该数值
- `包含文本` - 模糊匹配

点击"预览结果" → "导出Excel" → 完成！

---

### 方式3️⃣ 快速查询（查个人信息）
**适合快速查看某人的某个字段**

简单一句话格式：`姓名 字段名`
```
"张三 hrbp" → 显示张三的HRBP是谁
"李四 合同主体" → 显示李四的合同主体
"王五 离职日期" → 显示王五什么时候离职
```

秒级响应，不用打开Excel！

## 🚀 核心优势

### ✅ 智能识别，无需记忆准确列名
- 说"合同结束日期" = "劳动合同/协议结束日期"
- 说"hrbp" = "HRBP姓名"
- 说"部门" = "所属部门"

### ✅ 多条件组合筛选
- 部门 + 日期 + 状态 组合查询
- 支持精确匹配、模糊匹配、数值比较
- 智能处理部门名称（自动模糊匹配"电商部"、"电商事业部"）

### ✅ 智能排序和数量限制
- "最近5个" → 自动按日期降序 + 限制5条
- "最早入职的10名" → 自动按入职日期升序 + 限制10条
- 无需手动Excel操作

### ✅ 自动导出Excel文件
- 一键生成标准Excel格式
- 保留完整数据（所有列）
- 支持自定义导出字段

### ✅ 敏感数据自动保护
自动过滤敏感信息，保障数据安全：
- 身份证号、银行账号
- 电话号码、邮箱地址
- 家庭住址、紧急联系人

### ✅ 可视化Web界面
- 友好的操作界面，无需命令行
- 实时预览查询结果
- 支持多文件切换管理

## 💡 实际场景示例

### 场景1：领导临时要数据
**传统方式**：打开Excel → 筛选部门 → 复制粘贴 → 另存为 → 5分钟
**使用工具**：输入"找出电商部门的员工" → 点击下载 → 10秒搞定 ⚡

### 场景2：合同到期提醒
**传统方式**：手动看每一行日期 → 标记快到期的 → 导出名单 → 容易遗漏
**使用工具**：输入"合同3个月内到期的员工" → 自动筛选排序 → 不会遗漏 ✅

### 场景3：同事快速咨询
**传统方式**：打开Excel → Ctrl+F搜索 → 找到后告诉同事 → 2分钟
**使用工具**：输入"张三 hrbp" → 直接显示结果 → 5秒完成 🚀

### 场景4：离职人员分析
**传统方式**：筛选离职 → 排序日期 → 复制前10条 → 多个步骤
**使用工具**：输入"最近离职的10个人" → 一步到位 → 还能看到趋势 📊
