# Excel 数据提取工具 - 快速使用指南

> 一个基于AI的Excel智能查询工具，让HR数据查询变得简单高效！

## 🚀 5分钟快速上手

### 第一步：安装启动

```bash
# 1. 下载项目
git clone https://github.com/你的用户名/excel-data-extractor.git
cd excel-data-extractor

# 2. 启动后端（终端1）
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000

# 3. 启动前端（终端2）
cd frontend
npm install
npm run dev

# 4. 打开浏览器
访问 http://localhost:5173
```

### 第二步：配置AI（仅首次）

1. 点击右上角 **"配置API"** 按钮
2. 选择 **DeepSeek**（推荐，有免费额度）
3. 获取API Key：访问 https://platform.deepseek.com
4. 粘贴API Key并保存

✅ 配置完成！

---

## 💡 三种查询方式

### 方式1️⃣：字段筛选（精确查询）

**适合场景**：明确知道查询条件

**示例操作**：
```
所属部门：电商
劳动合同/协议结束日期：<2024-12-31
```
→ 点击"预览结果" → "导出Excel"

**语法说明**：
- 直接输入 = 精确匹配
- `<日期` = 小于该日期
- `>数字` = 大于该数值

---

### 方式2️⃣：快速查询（查个人信息）

**适合场景**：查某人的某个字段

**格式**：`姓名 字段名`

**示例**：
```
张三 hrbp          → 查张三的HRBP是谁
李四 合同主体      → 查李四的合同主体
王五 离职日期      → 查王五什么时候离职
```

---

### 方式3️⃣：AI智能查询 ⭐（最推荐）

**适合场景**：自然语言提问

**只需要这样问**：

```
找出最近离职的5个人
查询电商部门的员工
合同3个月内到期的员工有哪些
```

✅ **AI自动理解你的意图，直接导出Excel！**

## ⚠️ 常见问题

### Q: 提示"请先配置大模型API"？
**A**: 点击右上角"配置API"，填入DeepSeek的API Key

### Q: 查询结果不准确？
**A**: 
- AI自动识别意图，如："电商"会自动模糊匹配"电商部"、"电商事业部"
- 日期会自动排除无固定期限（2999年）

### Q: 导出的Excel缺少某些列？
**A**: 系统自动过滤敏感信息（身份证、银行卡、电话等），保护数据安全

## 🎯 小技巧

✨ **Tip 1**: AI查询支持数量限制

```

```

<2024-01-01  → 自动排除2999年（无固定期限）
